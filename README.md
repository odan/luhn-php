## Requirements

* PHP 8.1+

## Installation

```
composer require selective/luhn
```

## Usage

### Create a number

```php
<?php

use Selective\Luhn\Luhn;

$luhn = new Luhn();

echo $luhn->create('7992739871'); // 3
```

### Validate a number

```php
<?php

use Selective\Luhn\Luhn;

$luhn = new Luhn();

$luhn->validate('79927398713'); // true
$luhn->validate('79927398710'); // false
```
